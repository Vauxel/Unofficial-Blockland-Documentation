Welcome

Unnofficial Blockland Documentation is a community-driven documentation of the TGE scripting language, TorqueScript, as it is implemented in the lego-like sandbox game, Blockland.

However, this documentation does not encompass every function in Blockland's API; It only strives to document and provide information on the more fundamental functions and methods provided. For a complete, but less detailed, hierarchy and documentation of the functions, classes, and datablocks (etc) in Blockland's API, please visit the Blockland API Website provided by DontCare4Free.
